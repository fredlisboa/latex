\vspace{0.5cm}

\noindent\textbf{Observações:}
\begin{itemize}
    \item Cada patch panel possui \textbf{24 portas RJ-45} para cabeamento estruturado Cat5e/Cat6.
    \item Os pontos marcados como ``Carrinho'' referem-se a conexões em estações móveis dentro dos consultórios.
    \item Os pontos marcados como ``Bancada'' ou ``Balcão'' referem-se a conexões fixas em mobiliário.
    \item Os pontos marcados como ``WiFi TP Link Deco'' referem-se a access points da rede mesh sem fio.
    \item A numeração das portas segue a ordem física de cima para baixo no patch panel.
    \item Recomenda-se manter registro atualizado sempre que novos pontos forem ativados ou desativados.
\end{itemize}
